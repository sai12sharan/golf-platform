## **🎯 Complete User Flow (Step-by-Step)**

### **Step 1: Visit Homepage**
- Open http://localhost:3000
- See modern 3D landing page
- Click "Get Started"

### **Step 2: Create Account**
- Click "Don't have an account? Register"
- Fill in Name, Email, Password
- Click "Create Account"

### **Step 3: Subscribe to Plan**
- Click "Subscribe Now" card
- Choose Monthly ($9.99) or Yearly ($99.99)
- Click Subscribe

### **Step 4: Enter Your Score**
- Go to "Enter Scores"
- Set Course Rating (e.g., 72.0)
- Set Slope (e.g., 113)
- Select scores for all 18 holes:
  - **Hole 1-9** (Front 9)
  - **Hole 10-18** (Back 9)
- View your Stableford points automatically calculated
- Click "Submit Score"

### **Step 5: Check Leaderboard**
- Click "Leaderboard"
- See your ranking
- View total points and average

### **Step 6: View Draws**
- Click "Draws"
- Click "Create Draw" button
- See random monthly winners

### **Step 7: Support Charity**
- Click "Support Charity"
- Select a charity
- See impact statistics

### **Step 8: View Dashboard**
- Click "Dashboard"
- See all your stats
- Access all features

## **🐛 Troubleshooting**

### **Issue: "Cannot reach API"**
- ✅ Make sure backend is running (port 5000)
- ✅ Check .env file configuration
- ✅ Verify CORS settings

### **Issue: "Module not found"**
- ✅ Run `npm install` in client folder
- ✅ Clear `.next` cache
- ✅ Restart dev server

### **Issue: "Page not loading"**
- ✅ Check browser console for errors
- ✅ Verify http://localhost:3000 is accessible
- ✅ Check network tab in DevTools
